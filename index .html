Šta je Arduino?

Arduino je mikrokontrolerska platforma razvijena 2005. godine na Institutu za interaktivni dizajn u Ivreji, Italija. Open-source hardverska i softverska platforma koja omogućava upravljanje elektronskim komponentama kroz C/C++.

Platforma se zasniva na 8-bitnom Atmel AVR mikrokontroleru (popularni Uno) ili 32-bitnom ARM procesoru (napredniji Due). Programira se putem besplatnog Arduino IDE softvera (www.arduino.cc) koji radi na Windows, macOS i Linux sistemima.

Hardverska arhitektura

  • Digitalni I/O — HIGH (5V) ili LOW (0V)
  • Analogni ulazi — 0–5V → vrednosti 0–1023 (10-bit ADC)
  • PWM pinovi (~) — analogni izlaz digitalnim signalom
  • USB — komunikacija i upload programa
  • Napajanje — USB 5V ili eksterno 7–12V DC

Shield-ovi

Pločice koje se postavljaju direktno na Arduino i proširuju funkcionalnost bez lemljenja. Postoje shield-ovi za: WiFi, Bluetooth, Ethernet, motore, GPS, OLED i mnoge druge namene.

💡 S obzirom na veoma jednostavnu upotrebu i male, bezbedne napone (5V DC), Arduino može poslužiti kao odlično nastavno sredstvo na časovima TIK-a i informatike — bez ikakvih izmena nastavnih planova.

Sadržaj

LED dioda

Duža nožica = anoda (+), kraća = katoda (−). Otpornik 220Ω redno obavezno — bez njega dioda se uništava. Anoda → 220Ω → pin; Katoda → GND.

Taster

Pull-down 10kΩ između signal pina i GND. Otpušteno = LOW (0V), pritisnuto = HIGH (5V). Bez pull-down otpornika pin "pluta" i daje nepouzdana očitavanja.

Zujalica

Crvena/duža nožica (+) → digitalni pin (OUTPUT). Crna (−) → GND. Zvuk: tone(pin, frekvencija).

Ultrazvučni senzor HC-SR04

PinVeza
Vcc+5V
GNDGND
TrigDigitalni pin (OUTPUT)
EchoDigitalni pin (INPUT)

Foto-otpornik (LDR)

Analogni pin + pull-up 10kΩ ka +5V. Vrednosti: 0 (svetlo) → 1023 (mrak).

Servo motor

Crvena→5V, crna/braon→GND, žuta/narandžasta→PWM pin (~). Rotacija 0–180°. Biblioteka: Servo.h.

⚠️ Snažniji servo zahteva eksterno napajanje — spojiti masu eksternog napajanja sa GND Arduina!

RGB LED

4 nožice: R, G, B + zajednička katoda (→GND). Obavezno PWM pinovi. Uno PWM pinovi: 3, 5, 6, 9, 10, 11.

DHT11 / DHT22

Pin1→5V, Pin2→signal + pull-up 10kΩ, Pin3→slobodan, Pin4→GND. Biblioteka: DHT.h.

Relej

Vcc→5V, GND→GND, In→signal pin. NO (normalno otvoren): HIGH = uključen. NC (normalno zatvoren): inverzna logika.

LED matrica MAX7219

VCC→5V, GND→GND, DIN→pin12, CS→pin10, CLK→pin11. Biblioteka: LEDControlMS.h.

IR prijemnik

Leva nožica→digitalni pin, srednja→GND, desna→+5V. Biblioteka: IRremote.h. Prima 38kHz signale.

Dvoprogramski sistem

Kombinacija C# i Arduino znači dva potpuno odvojena programa:

  • C# program (na računaru) — WinForms GUI, prima korisnički unos, šalje komande na serijski port

Potrebno je napisati dva programa — u C# se definiše grafičko okruženje i šalju podaci na serijski port. S druge strane, Arduino prima te podatke i sprovodi određene akcije.

Korisnici Ultimate verzije Visual Studija mogu koristiti Visual Micro koji omogućava učitavanje i pisanje Arduino koda direktno u MS okruženju.

Princip PWM

Arduino digitalni izlazi daju samo 5V ili 0V. Za dimovanje LED ili upravljanje servo motorom potreban je napon između tih vrednosti. PWM brzo uključuje i isključuje signal simulirajući analogni napon. Ključni pojam: duty cycle — procenat vremena u kom je signal HIGH.

analogWriteDuty cycleEfektivni napon
00%0V
6425%~1.25V
12850%~2.5V
19275%~3.75V
255100%5V
PWM pinovi Uno: 3, 5, 6, 9, 10, 11 (označeni ~ na ploči). Mega: 2–13 i 44–46.
arduino / c++Fade + RGB mešanje boja
// Fade — postepeno dimovanje LED na pin 9
void loop() {
  for (int i=0; i<=255; i++) { analogWrite(9,i); delay(8); }
  for (int i=255; i>=0; i--) { analogWrite(9,i); delay(8); }
}

// RGB LED — R=9, G=10, B=11 (zajednička katoda)
// Žuta = R MAX + G MAX
analogWrite(9,255); analogWrite(10,255); analogWrite(11,0);
// Ljubičasta
analogWrite(9,180); analogWrite(10,0);   analogWrite(11,200);
// Bela = sve MAX
analogWrite(9,255); analogWrite(10,255); analogWrite(11,255);

P-01Arduino
Semafor
3 LED diode u semaforskom redosledu. Uvodni primer za digitalWrite() i delay().
+
Opis i kod

Tri LED diode (crvena pin 13, žuta pin 12, zelena pin 11) naizmenično svetle u pravom semaforskom redosledu. Svaka LED ima otpornik 220Ω ka GND. Odličan uvodni primer — demonstrira pinMode(), digitalWrite() i delay().

arduino / c++semafor.ino
// Semafor — crvena=pin13, žuta=pin12, zelena=pin11
void setup() {
  pinMode(11, OUTPUT); // zelena
  pinMode(12, OUTPUT); // žuta
  pinMode(13, OUTPUT); // crvena
}
void loop() {
  digitalWrite(11, HIGH); delay(3000); digitalWrite(11, LOW); // zelena 3s
  digitalWrite(12, HIGH); delay(1000); digitalWrite(12, LOW); // žuta 1s
  digitalWrite(13, HIGH); delay(3000); digitalWrite(13, LOW); // crvena 3s
  // crvena + žuta (prelaz nazad na zelenu)
  digitalWrite(13, HIGH); digitalWrite(12, HIGH);
  delay(1000);
  digitalWrite(13, LOW);  digitalWrite(12, LOW);
}

P-02Arduino
Svetleći prsten
8 LED dioda, brzina kruženja kontrolisana potenciometrom pomoću map(). Primer nizova i for petlji.

Opis

Osam LED dioda na pinovima 5–12. Potenciometar na A2 meri pauzu između paljenja. Funkcija map() mapira vrednosti 0–1023 na raspon pauze 30–500ms.

// Svetleći prsten — 8 LED pinovi 5-12, potenciometar A2
int pinovi[] = {5,6,7,8,9,10,11,12};

void setup() {
  for (int i=0; i<8; i++) pinMode(pinovi[i], OUTPUT);
}

void loop() {
  int pauza = map(analogRead(A2), 0, 1023, 30, 500);
  for (int i=0; i<8; i++) {
    digitalWrite(pinovi[i], HIGH);
    delay(pauza);
    digitalWrite(pinovi[i], LOW);
  }
}

P-03Arduino
Dan-noć
if-else u praksi: foto-otpornik (LDR) na A0, ako je mrak → uključi LED. Automatska uličnu svetiljka.
+
Opis i kod

LDR na analognom ulazu A0 sa pull-up 10kΩ ka 5V. LED na pin 13 (220Ω). Ako je vrednost LDR-a ispod praga (mrak), LED svetli. Prag se podešava zavisno od okruženja.

arduino / c++dan_noc.ino
// Dan-noć: LDR na A0, LED na pin 13
int prag = 500;

void setup() {
  pinMode(13, OUTPUT);
  Serial.begin(9600);
}
void loop() {
  int v = analogRead(A0);
  Serial.println(v); // praćenje u Serial Monitoru
  digitalWrite(13, v < prag ? HIGH : LOW);
  delay(100);
}

P-05Arduino
Kockica na 7-segmentnom displeju
Cifre 1-6 na 7-seg displeju. Zajednička anoda = obrnuta logika (LOW = svetli). Taster za zaustavljanje.
+
Opis i kod

Displej sa zajedničkom anodom: zajednički pin → 5V (220Ω). Segmenti a–g na pinove 2–8. Obrnuta logika: LOW = segment svetli, HIGH = ugašen! Taster na pin 12.

arduino / c++kockica_7seg.ino — zajednička anoda
// Zajednička anoda: LOW = svetli, HIGH = ugašen
// Segmenti a,b,c,d,e,f,g na pinove 2,3,4,5,6,7,8

    if      (c == 'R') ocekujemo = 'R';
    else if (c == 'G') ocekujemo = 'G';
    else if (c == 'P') ocekujemo = 'P';
    else {
      int val = (int) c;
      if      (ocekujemo == 'R') analogWrite(pinR, val);
      else if (ocekujemo == 'G') analogWrite(pinG, val);
      else if (ocekujemo == 'P') analogWrite(pinB, val);
      ocekujemo = ' ';
    }
  }
}

private void Form1_Load(...) {
  port = new SerialPort("COM3", 9600);
  port.DataReceived += new SerialDataReceivedEventHandler(port_DataReceived);
  port.Open();
}
void port_DataReceived(object sender, SerialDataReceivedEventArgs e) {
  string data = port.ReadLine();
  int v;
  if (int.TryParse(data.Trim(), out v)) {
    // Invoke jer DataReceived je na drugom threadu!
    Invoke((Action)(() => {
      lblVrednost.Text = "Vrednost: " + v;
      progressBar1.Value = Math.Min(v, 1023);
      bool crno = v < 500;
      picBox.BackColor = crno ? Color.Black : Color.White;
      lblBoja.Text     = crno ? "■ CRNA" : "□ BELA";
    }));
  }
}

🔌 KOMPONENTE

17 komponenti
💡 LED Dioda
Duža nožica = anoda (+). Otpornik 220Ω redno obavezno. Napon 1.8–2.2V, maks. 20mA. Kratka nožica / zarez = katoda → GND.
🔘 Taster
Pull-down 10kΩ ka GND. Otpušteno = LOW (0V). Pritisnuto = HIGH (5V). 4 nožice u 2 para.
🔔 Zujalica (Piezo)
Crvena (+) → dig. pin (OUTPUT). Crna (−) → GND. Koristiti tone(pin, freq) ili digitalWrite. Bez biblioteke.
📡 Ultrazvučni HC-SR04
Vcc→5V, GND→GND, Trig→dig.out, Echo→dig.in. Rastojanje = (t×340)/2. Biblioteka NewPing.
☀️ Foto-otpornik (LDR)
Analogni pin + pull-up 10kΩ ka +5V. Vrednosti: 0 (svetlo) → 1023 (mrak). Detekcija osvetljenosti.
🎛 Potenciometar
Srednji kraj → analogni ulaz. Spoljni → 5V i GND. Okretanjem: 0–5V → 0–1023. Razdelnik napona.
7️⃣ 7-segmentni displej
Zajednička anoda: zajednički pin → 5V, obrnuta logika (LOW = svetli). Zajednička katoda: pin → GND. Otpornik 220Ω.
⚙️ Servo motor
Crvena→5V, crna/braon→GND, žuta/narandžasta→PWM pin (~). Rotacija 0–180°. #include <Servo.h>. Ekst. napajanje za snažne.
🔴🟢 Dvobojna LED
3 nožice: srednja = zajednička katoda → GND. Leva = crvena, desna = zelena. Istovremeno paljenje = žuta. PWM za nijanse.
🌈 RGB LED
4 nožice: R, G, B + zajednička katoda (2. nožica) → GND. Obavezno PWM pinovi. Uno PWM: 3,5,6,9,10,11.
▦ LED matrica MAX7219
VCC→5V, GND→GND, DIN→pin12, CS→pin10, CLK→pin11. #include <LEDControlMS.h>. Lančanje matrica. Samo zajednička katoda.
📻 IR prijemnik
Leva nožica→dig. pin, srednja→GND, desna→+5V. Prima 38kHz IR. #include <IRremote.h>. Prva koristiti P-08 za HEX kodove.
📶 JY-MCU Bluetooth
Vcc→5V, GND→GND. TXD → razdelnik (1kΩ+2.2kΩ) → pin0. RXD → pin1. Osloboditi pinove 0 i 1 pri upload-u!
🔁 Relej
Vcc→5V, GND→GND, In→signal pin. NO: HIGH = uključen. NC: inverzna logika, za havarijski stop.
📟 LCD1602 + tasteri
Inicijalizacija: (8,9,4,5,6,7). Tasteri → različiti naponi na A0. Kontrast: plavi potenciometar. #include <LiquidCrystal.h>.
🌡 DHT11/DHT22
Pin1→5V, Pin2→signal + pull-up 10kΩ, Pin3→slobodan, Pin4→GND. DHT22 = precizniji. #include <DHT.h>.
👁 TCRT5000 optokapler
IR emiter + foto-tranzistor. Izlaz → analogni pin + pull-up 10kΩ. Crna podloga ≈ mala vrednost. Praćenje trake.
